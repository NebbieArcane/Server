# Nebbie Arcane
##Server code
* Getting started:
    * apt install vagrant
    * cp config_default.h config.h
    * ./getworld
    * vagrant up
    * vagrant ssh
    * cd /vagrant
    * make
    * cd mudroot
    * ./myst

The /vagrant folder in the vagrant machine is mounted from the folder where the vagrantfile resides and is synced 2-way
