# Nebbie Arcane
##Server code
* Getting started
    * FIRST TIME
        * apt install vagrant
        * ./getworld
        * vagrant up
        * vagrant ssh
        * cd /vagrant
        * autoreconf
        * automake --add-missing
        * ./configure
        * make
        * cd mudroot
        * ./myst
        * vagrant up
    * DAY BY DAY
        * vagrant up
        * vagrant ssh
        * make
        * cd mudroot
        * ./myst

The /vagrant folder in the vagrant machine is mounted from the folder where the vagrantfile resides and is synced 2-way
